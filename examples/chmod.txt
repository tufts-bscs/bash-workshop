Quick chmod Example!
CONTROL YOUR FILE PERMISSIONS...

     REMEBER
...................
   4 = READ = r
   2 = WRITE = w
   1 = EXECUTE = x
...................

1. Create a file called "CHMODEMO" and write something 
   in it (do manually)
2. Check the permissions (HINT: ls -l)
3. Using letters, give the group read and write access
4. Check the new permissions (chmod []+[] filename)
5. Using numbers take all access away (HINT: chmod +nums file)
6. Try to view your file 
7. Using numbers, give the owner full access and the group 
   and others no access 
8. Try to view your file
